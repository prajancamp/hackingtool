## 🚀 Installation

### Requirements

- Python 3.x
- Git
- Linux / Kali Linux (recommended)

### Quick Install

```bash
git clone https://github.com/Z4nzu/hackingtool
cd hackingtool
chmod +x install.sh
sudo bash install.sh
```

### Docker

```bash
docker build -t hackingtool .
docker run -it hackingtool
```

---

## 🖥️ Usage

```bash
sudo python3 hackingtool.py
```

Navigate through the menu using the numbered options. Each category contains a list of tools that can be automatically installed and launched.
